MV Artículos Florales

Sistema de inventario y gestión de clientes para artículos florales

Desarrollado por Edwin Sánchez

MV Artículos Florales es un sistema web diseñado para administrar el inventario de productos florales y facilitar la gestión de clientes. La plataforma permitirá visualizar los productos disponibles, controlar existencias y mantener organizada la información de los clientes.
